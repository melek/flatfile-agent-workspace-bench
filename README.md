# flatfile-agent-workspace-bench

Benchmark and evaluation artifacts for [flatfile-agent-workspace](https://github.com/melek/flatfile-agent-workspace) — a same-shape A/B that compares versions of the workspace template against earlier versions of itself across a fixed scenario set, and surfaces where three scoring rubrics disagree about whether a change helps.

This repository is a frozen archive. The benchmark produced the evidence behind the template's first public release; it is preserved here for reference rather than active development.

See [`bench/README.md`](bench/README.md) for the methodology, the scenarios, and how the runner works.
