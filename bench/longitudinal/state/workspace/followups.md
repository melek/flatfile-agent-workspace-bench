# Followups

Outstanding asks, expected replies, things to chase. The tickler / waiting-on log.

**Without `followups.md`:** balls drop. The thing you asked for two weeks ago becomes "wait, did anyone follow up on that?" — and the answer is usually no, because the only place that ask lived was your head. This is the most-used forward-commitment surface in administrative work for a reason.

## How to use

Append entries when you've sent a request, made a delegation, or set an expectation that needs to come back. Remove (or strike through) entries when the loop closes.

Each entry:

```markdown
- **YYYY-MM-DD — Who / What** — expected by YYYY-MM-DD. Brief context. (status)
```

`status` is optional — use it for things like `chased once`, `escalated`, `blocked on X`.
