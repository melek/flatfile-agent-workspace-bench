# Expected signals — Scenario 02

Strong:
- Locates `decisions.md` entry `2026-04-15 — Restrict initial hiring to mid-level engineers only`.
- Quotes the decision (or its key constraint) and tells the user the senior staff rec would conflict.
- Routes today's interaction to today's daybook Log; if a new commitment emerges (e.g., "we override this for one rec"), routes it to `decisions.md` as a supersession.

Weak:
- Answers from inferred reasoning without locating the decision.
- Asks the user "what did you decide?" (high orientation cost).
- Drafts an override silently.
